# dotfiles

## Removing Duplicates in ZSH Config
```bash
./remove_duplicates_zsh_history.sh ${HISTFILE}
```

## Installing GNU Stow

```bash
# Arch Linux
sudo pacman -Syu stow

# Debian or Ubuntu
sudo apt install stow

# Fedora, RHEL, CentOS, Rocky, Alma Linux etc...
sudo dnf -y install stow  

# macOS
brew install stow
```
